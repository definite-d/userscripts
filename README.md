## Installation

### Direct Install (GitHub)

Click any "Direct Install" link above. Your userscript manager will prompt you to install.

### GreasyFork

1. Visit the GreasyFork link for the script
2. Click "Install this script"

### OpenUserJS

1. Visit the OpenUserJS link for the script
2. Click "Install" button

## Auto-Update Sources

These scripts support auto-updates from multiple sources:

- **GitHub (Primary)**: Scripts auto-update from this repository
- **GreasyFork**: Set up with webhook to sync from GitHub
- **OpenUserJS**: Syncs from GitHub repository

To enable auto-updates, ensure your userscript manager has update checking enabled.
